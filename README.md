# AI-ML-PROJECT-FIRST-YEAR
THIS PROJECT WAS IN CIRRCULUM IT WAS ABOUT WILDLIFE CORRIDOR AND IDENTIFICATION AND SPECIES MOVEMENT PREDICTION
📌 Overview

The rapid expansion of urban areas, highways, and industrial zones has led to severe habitat fragmentation, increasing human–wildlife conflicts and road accidents. This project proposes an AI-driven wildlife corridor system that predicts safe and unsafe animal crossing zones using machine learning.

The model uses Random Forest Classification along with simulated animal movement data, sensor inputs, and drone imagery to improve habitat connectivity and support sustainable conservation planning.
